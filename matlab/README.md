## Name
Shimmer LSL

## Description
Signal recorder for Shimmer wristbands. It also include to possibility to visualize live stream, stream the data over LSL and save it to CSV file format.

This project is based on the resources, requirements and example downloaded from [Mathworks](https://www.mathworks.com/matlabcentral/fileexchange/43712-shimmer-matlab-instrument-driver)
